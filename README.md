# Smartlink Frontend Test

This repo is for doing Frontend Test from Smartlink (PT. Ada Ide Langsung Jalan)<br />

## Installation

Clone the repo.

```bash
git clone https://github.com/gaisuke/smartlink-fe-test.git
```

Install the package

```bash
npm install
```

Run project 

```bash
npm run serve
```

Compile and build for production

```bash
npm run build
```

## Tech Stack

- Vue
- Vuex
- Axios
