# smartlink-fe-test

This repo is for doing Frontend React Online Test from DOT Indonesia (PT. Digdaya Olah Teknologi)<br />

## Installation

Clone the repo.

```bash
git clone https://github.com/gaisuke/smartlink-fe-test.git
```

Install the package

```bash
npm install
```

Run project 

```bash
npm run serve
```

Compile and build for production

```bash
npm run build
```

- Vue
